# MEDICAL -Voice Assistant Chat bot

This is a  webhook implementation that gets Medical response from classification JSON (i.e. a JSON output of Api.ai /query endpoint) and returns a fulfillment response.

More info about Api.ai webhooks could be found here:
[Api.ai Webhook](https://docs.api.ai/docs/webhook)
